Html

超文本標記語言（英語：HyperText Markup Language，簡稱：HTML）是一種用於建立網頁的標準標記語言。常與CSS、JavaScript一起被眾多網站用於設計網頁、網頁應用程式以及行動應用程式的使用者介面。

JavaScript

JavaScript（通常縮寫為JS）是一種高級的、解釋型的程式語言。JavaScript是一門基於原型、函數先行的語言，是一門多範式的語言，它支持物件導向程式設計，命令式編程，以及函數式編程。

TypeScript
